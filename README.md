### 安装

```tex
1 将前端 mahjong-web 项目编译打包后的build下所有文件拷贝至项目ui目录

2 再进行 go build 编译成二进制文件

3 启动 ./mahjong
```

### 快速启动

```
windows 下载文件 mahjong.exe 直接执行

mac/linux 下载文件 mahjong 直接执行

浏览器访问 http://localhost:7070
```
